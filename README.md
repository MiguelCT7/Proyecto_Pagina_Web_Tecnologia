## Descripción

Este proyecto consiste en el desarrollo de una página web informativa sobre la tecnología moderna, abordando temas como:

- Inteligencia Artificial  
- Redes de Computadoras  
- Robótica  
- Historia de la Tecnología  

## Tecnologías utilizadas

- HTML5  
- CSS

## Características

- Diseño moderno y responsivo básico  
- Uso de estilos externos mediante archivo `.css`  
- Navegación interna con anclas  
- Tarjetas informativas (cards)  
- Galería de imágenes  
- Formulario de contacto  
- Tabla informativa histórica  
- Animaciones con `@keyframes`  
- Transiciones y efectos `hover`  

## Elementos destacados

Se implementaron animaciones utilizando CSS, aplicando conceptos como:

- `transition`
- `@keyframes`
- `animation`
- `transform`

Además, se integró una figura básica animada (robot) utilizando elementos `<div>` y estilos CSS.
